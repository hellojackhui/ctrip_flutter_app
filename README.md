# ctrip_flutter_app

A ctrip app written by flutter

## Build Setup
```git
// clone project
git clone https://github.com/hellojackhui/flutter_ctrip_app

// active project
cd ctrip_flutter_app
flutter run
```

## Environment (Android)
    - Java 1.8+
    - Flutter Environment 1.0+
    - Android SDK
    - Android Emulator
